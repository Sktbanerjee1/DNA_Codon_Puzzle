For the teacher This game uses the real standard genetic code — all 64 codons coding for 20 amino acids plus 3 stop codons (TAA, TAG, TGA). Words are spelled in amino acid one-letter codes. To keep things accessible for middle schoolers, mutations are restricted to single-base substitutions, which lets students explore three big concepts: degeneracy — most amino acids have multiple synonymous codons, so the same word can be encoded different ways each round; silent mutations — substitutions that produce a synonymous codon, leaving the protein unchanged; missense mutations — substitutions that swap one amino acid for another (sickle cell anemia is a real example); and nonsense mutations — substitutions that create a premature stop codon, truncating the protein. Frameshifts (insertions and deletions) are intentionally omitted — they're conceptually heavier and worth a separate lesson once students are comfortable with the codon table.
